# <a name="q-libraries"></a>Q# kitaplıkları

Q# standart kitaplığında, kuantum programları geliştirmek için kullanılan işlemler, işlevler ve kullanıcılar tarafından tanımlanan türler yer alır. Bu kitaplıkta, standart kitaplığın üzerine yapılandırılmış üç ek kitaplık olan kimya, sayısal ifade ve makine öğrenmesi kitaplıkları da bulunur.

Q# kitaplıkları hakkında daha fazla bilgi için bkz. <xref:microsoft.quantum.libraries>.
